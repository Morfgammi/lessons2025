translate

translate(X, Y)

Зсуває елемент одночасно по осях X (вправо) і Y (вгору).

translateX

Зсуває елемент лише по осі X — вправо або вліво.

translateY

Зсуває елемент лише по осі Y — вгору або вниз.

scale

scale(x, y)

Масштабує елемент

scaleX(x)

Змінює ширину.

scaleY(y)

Змінює висоту.

rotate

rotate(45deg)

Обертання елемента у 2D-площині.

rotateX(45deg)

Обертання навколо осі

rotateY(45deg)

Обертання навколо осі

skew

skew(x, y)

Нахиляє елемент по обох осях X і Y.

skewX(x)

Нахиляє елемент лише по осі X

skewY(y)

Нахиляє елемент лише по осі Y
