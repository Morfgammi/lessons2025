Відображення (Display)

Текст для прикладу (властивість display: block)Поведінка: Елемент відображається як блочний, завжди починаючи з нової строки.
Текст для прикладу (властивість display: inline-block;)Елемент поводиться як строковий зовні (не створює новий рядок), але як блочний всередині.
Текст для прикладу (властивість display: inline;) Елемент відображається як строковий, не створюючи нового рядка.

Розміри

width / height висота та ширина коробочки width(ширина) : 100px; height(висота): 150px;
max-width — це властивість у CSS, яка встановлює максимально допустиму ширину елемента. Вона не дає елементу розширюватися більше вказаного значення, що робить її корисною для створення адаптивних зображень та текстів, які не "ламають" верстку при зміні розмірів екрана.
min-width — це властивість CSS, яка встановлює мінімальну ширину для елемента, запобігаючи її зменшенню нижче заданого значення. Якщо розмір екрану стає меншим, ніж встановлений min-width, ширина елемента залишається незмінною, і для перегляду з'являється горизонтальна смуга прокрутки.
min-height — це властивість у CSS, яка встановлює мінімальну висоту для елемента, не дозволяючи їй стати меншою за вказане значення, незалежно від вмісту або розміру вікна браузера. Якщо обчислена висота елемента виявляється меншою за min-height, браузер застосує саме min-height, гарантуючи, що елемент не буде занадто коротким.
max-height — це властивість у CSS, яка встановлює максимальну висоту елемента, запобігаючи його надмірному зростанню по вертикалі, навіть якщо вмісту стає більше, ніж дозволено властивістю height. Це дозволяє створювати адаптивний дизайн, де елемент розтягується до певної межі, а потім його вміст може ховатися або переноситися.

Відступи зовнішні внутрішні

Padding в CSS – це властивість, яка створює внутрішній відступ навколо вмісту елемента, між цим вмістом і його кордоном (рамкою). Вона використовується для збільшення простору між текстом або іншим контентом та межами елемента, покращуючи читабельність та візуальну привабливість дизайну.
  • padding-top: відступ зверху
  • padding-right: відступ справа
  • padding-bottom: відступ знизу
  • padding-left: відступ ліворуч
У CSS, margin – це властивість, яка визначає зовнішні відступи (поля) навколо елемента, створюючи простір поза його границею і відштовхуючи інші елементи, що знаходяться поруч Контроль сторін: Ви можете задати відступ для кожної сторони окремо, використовуючи властивості:
  • margin-top: відступ зверху
  • margin-right: відступ справа
  • margin-bottom: відступ знизу
  • margin-left: відступ ліворуч
